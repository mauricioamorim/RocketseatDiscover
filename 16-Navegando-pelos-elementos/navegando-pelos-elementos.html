Navegando pelos elementos

Veja como navegar pelos elementos

IMPORTANTE observer que os metodos com "Element" (firstElementChild, nextElementSibling e etc) retornam somente os elementos HTML que existem. Já os metodos sem "Element" (firstChild, nextSibling e etc) retornam outras coisas como "text" quando ha uma quebra de linha em branca no código

perentElement e parentNode

navegação pelos elementos pais

parentNode
Retorna o nó pai
ParentElement
Retorna o elemento pai

childNodes, firstElementChild, lastElementChild

navegação pelos elementos filhos

childNodes
Retorna um nodeList com elementos filhos(linhas em branco são consideradas como text)
firstElementChild
Retorna o primeiro elemento filho
lastElementChild
Retorna o ultimo elemento filho

nextElementSibling e previousElementSibling

navegação pelos elementos irmãos (Sibling significa irmão)

previousElementSibling
Retorna o elemento irmão anterior
nextElementSibling
Retorna proximo elemento irmão
