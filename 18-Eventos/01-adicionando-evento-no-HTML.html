Adicioando evento pelo javascript

Tudo que tem ação é um dispara um evento(click, rolagem, pressionar teclas e etc)

Eventos se iniciam logo após uma ação

Adicionando evento por tag HTML

basta adicionar o atributo on[nome-do-evento] na tag HTML

Adicionando evento pelo javascrit

É possível adicionar o evento no JS pelo metodo addEventListener([nome-do-evento], callback)

Argumernto Event

Todo evento quando disparado passa como argumento o objeto Event

exitem categorias do objeto Event de acordo com o evento disparado

dentro do objeto é possível ter informações como: qual q tecla apertada, o código dela e etc.
