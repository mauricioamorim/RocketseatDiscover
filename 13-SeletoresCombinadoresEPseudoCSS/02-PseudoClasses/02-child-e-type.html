first-child

Funciona somente no primeiro elemento especificado

Se o elemento especificado for o segundo filho não irá aplicar o css

nth-of-type()

nth-child()

Seleciona o elemento de acordo com a ordem passada e indiferente se existem outros tipos de elementos antes dele. Mas a ordem tem que bater com o tipo de elemento selecionado

Se descomentar o 'li:first-child' vai parar de funcionar
