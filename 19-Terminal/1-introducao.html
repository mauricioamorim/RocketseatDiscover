Introdução ao Terminal

introdução ao conceito e terminologias

GUI
Inteface grádica de usuário. Interface geralmente mostra icones, janelas e etc...
CLI
Interface de linha de comando. Iterface que exibe apenas linhas de comando.
Shell ou interpretador
Executa os comandos, Conversa como Kernel
Kernel
Responsável por conversar como hardware da maquina
Bash
Shell padrão no ambiente linux

Sintaxe

ls -la /usr/bin
Command (-la)
Programa que vc está rodando
Option(s)
Mudar o comportamento. Explicar ao comando como deve funcionar. Geralmente começa com um traço, mas pode ser q tenha tambem dois traços. Geralmente cada letra representa um comando ("l" = list e "a" = todos os arquivos ocultos). Pode formar combos juntando letras (-la) ou serem escritas independentes (-l -a), mas quando tiver dois traços tem q ser separado. Não importa a ordem das letras
Argumentos
Onde o comando irá operar. Caminho do arquivo ou diretórios.
