Obtendo informações dos arquivos
Nesta aula, falaremos sobre os comandos file e stat, ambos utilizados para obeter maiores informações dos arquivos, utilizados da seguinte forma: file NomeDoSeuArquivo.txt O comando file é utilizado para obter informações gerais, como formato e informações do tipo de arquivo. já o comado stat é utilizado, ele mostra a permissão que se tem no arquivo, o dono e o grupo, tamanho, data de última modificação outras informações úteis. É usado da seguinte forma: stat NomeDoSeuArquivo.txt
