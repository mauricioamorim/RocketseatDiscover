Editando arquivos com nano e vim
Nesta aula, veremos como editar arquivos com o nano, que é um editor de texto, porém, você não pode usar seu mouse, para navegar usam-se as setinhas do teclado, e embaixo fica a legenda do que pode ser feito, através da tecla Ctrl e mais alguma letra, por exemplo a combinação Ctrl + O Salvaria esse arquivo, mas antes seria perguntado se deseja salvar o arquivo com o mesmo nome, caso responda que não, será mostrada uma tela para alterar o nome do arquivo. Para fechar o nano, o atalho é o Ctrl + X O vim é muito mais complexo que o nano, e pode-se inserir plugins nele, tornando-o mais poderoso. Para sair do vim, usa-se o comando :q Porém, se tiver feito alguma alteração, ele vai precisar ou do comando :wq, para confirmar, escrever essas mudanças e sair, ou do comando :q!, que cancela as alterações
