Instalando dependências globais

Caso vc queira, vc pode instalar o pacote na sua máquima deixando ele disponível num nível global (-g)

npm install cfonts -g

Obs. quando vc instala de maneira global a dependencia não vai aparecer no package.json e a pasta node_modules não aparece no projeto.

Caso vc querira ver em que pasta foi instalado digite o comando npm root -g

Para desinstalar o pacote digite o comando npm uninstall cfonts -g
