Instalando dependências

NPMinstall

Para instalar uma dependência ao seu projeto basta digitar npm install

exemplo:

npm install cfonts

Após instalar essa dependência ficará registrada no arquivo package.json

{ "name": "NodeJS", //Não pode ter o mesmo nome no npm "version": "1.0.0", "description": "", "main": "index.js", //o arquivo principal do pacote "scripts": { "test": "echo \"Error: no test scpecified\" && exit 1" }, "keywords": [ "aprendendo", "nodejs" ], "author": "Mauricio Amorim", "license": "ISC", "dependencies": { "cfonts": "^2.9.1" } }
