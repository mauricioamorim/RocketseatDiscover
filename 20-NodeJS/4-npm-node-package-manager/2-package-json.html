Package.json

O que é o arquivo package.json?

Package.json é um arquivo do tipo JSON

exemplo:

{ "name": "NodeJS", //Não pode ter o mesmo nome no npm "version": "1.0.0", "description": "", "main": "index.js", //o arquivo principal do pacote "scripts": { "test": "echo \"Error: no test scpecified\" && exit 1" }, "keywords": [ "aprendendo", "nodejs" ], "author": "Mauricio Amorim", "license": "ISC" }
