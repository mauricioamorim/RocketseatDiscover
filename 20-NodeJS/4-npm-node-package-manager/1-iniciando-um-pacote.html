Iniciando um pacote

Para iniciar um pacote vc deve digitar npm init no terminal e depois responder as seguintes perguntas:

O nome

A versão

A descrição

Arquivo inicial(entry pont)

Comando de teste(test comand)

Reppositorio git

Palavras chave

Autor

Tipo de licença

Iniciando um pacote automaticamente

Para iniciar um pacote sem ter que responder todas as perguntas basta vc deve digitar npm init -y(-y = yes para tudo) no terminal
