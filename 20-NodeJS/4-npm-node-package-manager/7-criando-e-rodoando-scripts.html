Como vc pode ver, dentro do package.json tem uma propriedade chamada "script" e dentro dela vc encontra a propriedade "start".

Para rodar o script "index.js" basta vc digitar no terminal npm run start

Obs. o termo "start", por ser uma palavra especial, não precisa ter o comando "run". Mas qualquer outro termo precisa
