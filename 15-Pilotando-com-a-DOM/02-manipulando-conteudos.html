Manipulando Conteúdos

Como manipular conteúdo do DOM

Aqui vc vai ver algumas maneiras de manipular o contúdo HTML através do DOM

textContent

altera o texto do conteúdo

Não permite colocar tags HTML

innerText

Troca o texto interno do HTML

Não permite colocar tags HTML

innerHTML

Troca o conteúdo do elemento

Permite colocar tags HTML

value

altera o valor de um input

tambem pode ser usado para pegar o valor do input

Atributos

cria um atributo com valor

altera ou pega o valor de atributo do elemento
